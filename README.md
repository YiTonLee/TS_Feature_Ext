# TS_Feature_Ext
This repository provides code and report archive of individual research: Deep Learning Approach in Time Series Forecasting: Literature Review and Extension on Feature Extraction.

The animation of N-BEATS model structure in the presentation slide is made using Manim, which is the Python package used by the 3Blue1Brown Channel to create videos explaining mathematical and machine learning concepts.

The experiment used the Exchange rate daily dataset from Time Series Library (TSLib).
